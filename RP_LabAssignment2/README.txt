USAGE:
To visualize the Light-Up Puzzle solution:
1. Open the terminal.
2. Navigate to the directory containing the puzzle files.
3. Execute the command: show <filename.txt>
   Replace <filename.txt> with the actual file name, e.g., "show dom01.txt".
